Assistant de Configuration

1. Configuration générale du site

Téléchargez d'abord le "Starter pack" du site. Il contient tous les fichiers HTML, CSS, JavaScript et les icônes par défaut.

Télécharger le Starter pack

2. Configuration Popote (Articles & Seuil de Dette)

Définissez le nom et le prix de chaque article de consommation. Ces valeurs seront utilisées dans l'interface Popote et dans le script Apps Script.

Seuils de Dette (€)

3. Création de la feuille de calcul

Le site utilise une feuille Google Sheets pour la gestion de la popote et du merch.

Étape A : Ouvrez le modèle ci-dessous. Pensez à l'ouvrir dans un nouvel onglet pour ne pas perdre la configuration en cours.

Ouvrir le modèle

Étape B : Une fois la feuille ouverte, faites une copie (Fichier > Faire une copie) sur votre propre compte Google. Remplissez les champs de chaque onglet sans toucher aux cases bleues.

Étape C : Partagez la feuille en mode "Tous les utilisateurs disposant du lien" avec le rôle "Lecteur".

4. Adaptation du script Apps Script

Nous allons adapter le script pour votre feuille. Téléchargez d'abord le fichier all.gs fourni dans le dépôt GitHub. Il doit contenir les marqueurs SPREADSHEET_ID_PLACEHOLDER et POPOTE_PRICES_PLACEHOLDER.

5. Déploiement du script

Maintenant, vous devez déployer le script sur Google Apps Script. Voici comment faire :

  1. Rendez-vous sur Google Apps Script et créez un "Nouveau projet".
  2. Collez le code adapté dans l'éditeur (depuis l'étape précédente).
  3. Cliquez sur "Déployer" > "Nouveau déploiement".
  4. Sélectionnez "Application web" et assurez-vous que l'accès est défini sur "Tout le monde".
  5. Cliquez sur "Déployer" et copiez l'URL de l'application web fournie.

6. Télécharger votre configuration

Votre site est configuré ! Téléchargez le fichier ZIP ci-dessous contenant tous les fichiers à jour dont vous avez besoin :

  • all.json (Config mise à jour)
  • popote.html (Rendu dynamique)
  • all.gs (Script backend mis à jour)

7. Finalisation du déploiement

C'est la dernière étape ! Votre site est prêt à être publié. Utilisez les fichiers que vous venez de télécharger pour mettre à jour votre dépôt GitHub. Vous pouvez trouver tous les fichiers de base nécessaires pour votre site à l'adresse ci-dessous.
