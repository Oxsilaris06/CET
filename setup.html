Télécharger le Starter pack

2. Création de la feuille de calcul

Le site utilise une feuille Google Sheets pour la gestion de la popote et du merch.

Étape A : Ouvrez le modèle ci-dessous. Pensez à l'ouvrir dans un nouvel onglet pour ne pas perdre la configuration en cours.

Ouvrir le modèle

Étape B : Une fois la feuille ouverte, faites une copie (Fichier > Faire une copie) sur votre propre compte Google. Remplissez les champs de chaque onglet sans toucher aux cases bleues.

Étape C : Partagez la feuille en mode "Tous les utilisateurs disposant du lien" avec le rôle "Lecteur".

4. Déploiement du script

Maintenant, vous devez déployer le script sur Google Apps Script. Voici comment faire :

  1. Rendez-vous sur Google Apps Script et créez un "Nouveau projet".
  2. Collez le code adapté dans l'éditeur.
  3. Cliquez sur "Déployer" > "Nouveau déploiement".
  4. Sélectionnez "Application web" et assurez-vous que l'accès est défini sur "Tout le monde".
  5. Cliquez sur "Déployer" et copiez l'URL de l'application web fournie.
